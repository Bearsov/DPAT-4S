### Made with Python 3.13.5
###  To install requirements run:
`pip install -r requirements.txt`

### Short instruction how to use:
1. Run all cells in presented order (to init everything properly)
2. After starting the last cell you will see a plot with sliders and few checkboxes
3. Move sliders to change input params (graph will update automatically)
4. Checkboxes 'Show noise' and 'Show filter' will hide/unhide Noise/Filter lines
5. 'Reset' button sets all params to default
